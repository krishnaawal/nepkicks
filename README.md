# Nepkicks Brilliant Shoes

Production-ready single product ecommerce landing page built with Next.js 15, React, TypeScript, Tailwind CSS, Node.js API routes, Resend email automation, Google Sheets order storage, and a secure admin dashboard.

## Run Locally

```bash
npm install
npm run dev
```

Open `http://localhost:3000`.

Admin dashboard: `http://localhost:3000/admin`

Full beginner setup instructions are in `SETUP_GUIDE.md`.
